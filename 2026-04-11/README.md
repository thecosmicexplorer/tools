# AWS Secrets Manager SSRF Scanner

## Description
This tool scans for the presence of AWS Secrets Manager endpoints and identifies potential server-side request forgery (SSRF) vulnerabilities by sending carefully crafted probes to the service. This is useful for pentesters, security researchers, and bug hunters targeting cloud-related services.

AWS Secrets Manager SSRF vulnerabilities can lead to exposure of sensitive credentials, metadata, or internal resource access.
