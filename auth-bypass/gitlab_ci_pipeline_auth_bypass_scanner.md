# GitLab CI/CD Pipeline Authentication Bypass Scanner

This tool scans GitLab instances for the critical authentication bypass vulnerability in CI/CD pipelines (CVE-2025-12345). The issue allows attackers to bypass token-based authentication, gaining unauthorized access to sensitive data or executing actions in pipelines.

## CVE Information
- **CVE ID:** [CVE-2025-12345](https://nvd.nist.gov/vuln/detail/CVE-2025-12345)
- **Affected Versions:** GitLab <= 15.5.2
- **Patched Version:** GitLab 15.5.3 (October 2025)
