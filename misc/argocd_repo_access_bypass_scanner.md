# ArgoCD Repository Access Bypass Vulnerability Scanner (CVE-2025-41234)

This tool detects ArgoCD instances vulnerable to the repository access bypass vulnerability (CVE-2025-41234). The vulnerability allows unauthenticated attackers to bypass repository access permissions and potentially read sensitive files, including source code and secrets.

## Affected Versions
ArgoCD versions **< 2.9.0** are vulnerable. The issue was fixed in ArgoCD 2.9.0 (June 2026).
